# aaronlmackenzie.github.io

Personal portfolio site for Aaron Mackenzie — Network Engineer.

Because this repo (`aaronlmackenzie.github.io`) does not match the account
username (`aaronlmack`), GitHub Pages serves it as a **project page** at:

**https://aaronlmack.github.io/aaronlmackenzie.github.io/**

(For the shorter `https://aaronlmack.github.io` URL, the repo would need to be
named `aaronlmack.github.io`.)

## Files
- `index.html` — the page
- `style.css` — all styling
- `assets/profile.jpg` — profile photo

## Enable GitHub Pages (one-time)
1. Go to the repo → **Settings → Pages**.
2. Under **Build and deployment → Source**, choose **Deploy from a branch**.
3. Branch: `main`, folder: `/ (root)`. Save.
4. Wait ~1 minute, then open the URL above.

## Updating later
Edit the files, then:
```bash
git add .
git commit -m "Update site"
git push origin main
```
